## 简单计算题

### poj-2750-鸡兔同笼
脚的总数：N
1. N为奇数，直接输出“0 0”；
2. N为4的倍数，输出N/4和N/2；
3. N不为4的倍数，输出N/4+1和N/2；

### poj-1657-棋盘上的距离
x = 起始位置与终止位置在水平方向上的距离（如f-e=1，在每步一格的情况下，f到e需要一步）
y = 起始位置与终止位置在竖直方向上的距离（如5-2=3，在每步一格的情况下，5到2需要三步）

王：x,y中的最大值
后：同行/同列/x=y  --> 1
    其他情况  --> 2
车：同行/同列  --> 1
    其他情况  --> 2
象：x-y的绝对值不是偶数  --> Inf
    x=y  --> 1
    其他情况  --> 2

注意！！是Inf，不是lnf也不是inf。
